## Time Complexity

Time complexity measures how runtime grows with input size n.

### Common Complexities
- O(1)  → Constant
- O(n)  → Linear
- O(log n) → Logarithmic
- O(n log n) → Linearithmic
- O(n²) → Quadratic

### Rules
1. Ignore constants
2. Ignore lower-order terms
3. Sequential loops add
4. Nested loops multiply
